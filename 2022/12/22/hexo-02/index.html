配置 Github

使用邮箱注册 GitHub 帐号并登录：Github 

点击右上角加号图标，选择 New repository 创建一个新仓库，仓库名为：用户名.github.io，该 用户名 使用自己的 GitHub 帐号名称代替。

打开 GitHub_Settings_keys 页面，点击 New SSH key 按钮，Title 处为本台计算机取一个名字，然后将刚刚复制的 id_rsa.pub 内容粘贴进去，最后点击 Add SSH key。
